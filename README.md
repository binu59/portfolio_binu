React + TypeScript + Tailwind (Vite) site showcasing skills, projects, and career goals as an aspiring Data Scientist / ML Engineer.

## Quick start

```bash
npm install
npm run dev        # start dev server
npm run build      # production build
npm run preview    # preview production build
```

## Content to update
- Projects: add dataset source, metrics (MAE/RMSE or key insights), GitHub links, and live demos in `src/App.tsx`.
- Resume link: replace `#resume-link` in `socials.resume` with your URL.
- Certifications: replace placeholder bullets in `certifications` list.
- Education/experience: adjust text and add new entries as needed.

## Deploy
- **Vercel**: import the repo, set framework to Vite, and deploy (build: `npm run build`, output: `dist`).
- **GitHub Pages**: `npm run build` then push `dist` via `gh-pages` or Pages action.

## Tech stack
- React 19 + TypeScript, Vite, Tailwind CSS
- Dark theme palette with cyan/violet accents
